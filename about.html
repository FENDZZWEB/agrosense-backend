About System

AgroSense

Smart Irrigation Monitoring System · v2.1

Tentang Aplikasi

AgroSense adalah platform dashboard cerdas yang dirancang untuk membantu pemantauan iklim mikro dan kelembaban tanah sawah tadah hujan secara real-time. Sistem ini dilengkapi dengan integrasi model kecerdasan buatan (LSTM) untuk memprediksi kebutuhan pengairan lahan berdasarkan data cuaca historis dan prediksi AccuWeather.

Teknologi Utama

  • Frontend: HTML5, Tailwind CSS, Chart.js, React (untuk komponen tabel)
  • Backend & Database: Firebase Realtime Database & Firebase Hosting
  • AI / Machine Learning: Python, TensorFlow (Model LSTM)
  • Hardware: ESP32 TTGO T-Call (GPRS/WiFi), Sensor DHT22, Sensor FC-28

Tim Pengembang

Dikembangkan untuk keperluan penelitian akademik terapan dalam meningkatkan produktivitas pertanian sawah tadah hujan melalui pendekatan Internet of Things dan Machine Learning.
